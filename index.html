使用说明

CineIA GUI 是一个用于将 IMF IAB (.mxf) 转换为 DCP IAB (.mxf) 的工具。

  1. 输入文件: 点击“选择文件”或直接将 .mxf 文件拖入输入框。
  2. 输出路径: 系统会自动根据输入文件生成输出路径，您也可以点击“选择路径”手动更改。
  3. 参数设置:
    • No Copy Preamble: 不复制前导符 (用于修复某些可能有问题的流)。
    • Force Dolby: 强制应用杜比约束 (仅在确实且清楚自己在做什么时勾选)。
    • Channels: 默认 10 (对应 7.1.2 布局)。
    • Objects: 默认 118。
  4. 开始转换: 点击按钮开始，下方进度条会显示实时进度。

Instructions

CineIA GUI is a tool for converting IMF IAB (.mxf) to DCP IAB (.mxf).

  1. Input File: Click "Select File" or drag & drop an .mxf file.
  2. Output Path: Automatically generated based on input. Click "Select Path" to change.
  3. Parameters:
    • No Copy Preamble: Do not copy preamble values (fixes some bad streams).
    • Force Dolby: Enforce Dolby constraints (Use with caution).
    • Channels: Default 10 (7.1.2).
    • Objects: Default 118.
  4. Start: Click "Start Conversion" to begin. Progress is shown below.
